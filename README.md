# Blockchain-Todo-App

To use app you need:
- Metamask to be installed
- to be connected to Rinkeby test network with Metamask
- and have some eth on wallet, you can get [here](https://faucets.chain.link/rinkeby)

Technology stack <br />
  1. React, Typescript, Mui, ether.js <br />
  2. Solidity,hardhat
